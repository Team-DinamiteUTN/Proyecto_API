# API-Project

This is a simple API to manage users & Cubes.

# Install

## Step 1
To install all the dependencies for the project.
```bash
npm install
```
## Step 2
### Run project

```bash
npm run dev
```

# How to use it.
Test with postman with the following routes:
- http://localhost:3000/users
- http://localhost:3000/cubes

# Schemas
## User
{
    "name": "myname",
    "password": "mypassword"
}
## Cube
{
    "Color": "examplecolor",
    "Weight": 0
}
